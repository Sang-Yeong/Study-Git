## git의 목적2: 백업
	- 원격 저장소

	- pull: 원격 --> 지역

## git의 종류
- 제품군
- TortoiseGit: 윈도우 탐색기에 기생해서 사용되는 프로그램
- SourceTree: 복잡 && 자유로움, 버전마다 속해있는 파일들을 보여줌
- git original: 명령어를 통해 깃을 제어
